Documentacao API

Referencia das chamadas utilizadas pelos servicos auth.service e brapci-api.service.

AuthService

Metodo HTTP Endpoint Parametros Resultado Esperado
checkSession() GET /auth/me Sem body. { authenticated: boolean, user: User | null }
login(payload) POST /auth/login ou /auth/login-email { username: string, password: string } { authenticated: boolean, user: User | null }
register(payload) POST /auth/register { name: string, username: string, email: string, password: string } { authenticated: boolean, user: User | null }
resendPassword(payload) POST /auth/forgot-password { email: string } { success?: boolean } (service converte para boolean)
logout() POST /auth/logout {} 204 No Content

BrapciApiService

Metodo HTTP Endpoint Parametros Resultado Esperado
get(endpoint, params) GET {brapciApiBaseUrl}/{endpoint} endpoint?: string
params?: Record<string, string | number | boolean> 		T generico (retorno da API Brapci)
search(query) GET {brapciApiBaseUrl}?q={query} { query: string } T generico (resultado de busca)

Base de auth definida por buildAuthUrl() em auth.service. Base Brapci definida em API_CONFIG.brapciApiBaseUrl.
